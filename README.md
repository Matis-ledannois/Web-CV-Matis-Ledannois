# 🤖 Interactive Robotic CV – Matis Ledannois

Interactive online CV built as a **single-page web application** to showcase my skills in **Software Development, Data Analysis, and Robotics Engineering**.

🔗 Designed to provide recruiters with a dynamic and engaging way to explore my profile.

## 🚀 Live Overview

This project is an interactive dashboard-style CV featuring:


- 📌 Sticky navigation bar
- 🕒 Expandable professional experience timeline
- 📊 Dynamic radar chart for technical skills
- 🌍 Visual language proficiency bars
- 🎨 Modern UI built with Tailwind CSS

## ✨ Features

### 🧑‍💼 Interactive Timeline
- Dynamically generated professional experiences
- Click-to-expand job descriptions
- Clean vertical timeline layout

### 📊 Skills Radar Chart
- Built with Chart.js
- Filterable categories:
  - Programming
  - Data & BI
  - Tools & Methodology
- Real-time chart updates via JavaScript

### 🌐 Language Proficiency
- Visual progress bars
- Clear and intuitive representation

### 📱 Responsive Design
- Fully responsive
- Optimized for desktop and mobile devices
